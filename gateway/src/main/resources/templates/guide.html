Guía de Uso

La Plataforma

Bienvenido al proyecto de TFG de Tomás Darquier, una Plataforma de Generación de Código y Arquitecturas de Microservicios, en otras palabras, una herramienta innovadora que te permite crear y gestionar arquitecturas de microservicios de manera visual e intuitiva. Esta plataforma está diseñada para simplificar el proceso de diseño y desarrollo de sistemas distribuidos, permitiéndote concentrarte en la lógica de negocio mientras la plataforma se encarga de brindarte cimientos funcionales y testeados, donde construir tu sistema.

Mediante una interfaz drag-and-drop, podés construir arquitecturas complejas arrastrando y conectando componentes predefinidos, cada uno representando un servicio específico con su propia funcionalidad y características únicas.

¿Cómo Funciona?

1

Selección de Componentes

Arrastra los componentes necesarios desde la barra lateral izquierda hacia el área de diseño. Cada componente representa un microservicio específico con funcionalidad predefinida.

2

Configuración

Personaliza cada componente según tus necesidades ajustando sus parámetros de configuración. Puedes definir aspectos como la persistencia, seguridad y comunicación entre servicios.

3

Conexiones

Establece las relaciones entre los diferentes microservicios conectándolos en el canvas. Las conexiones representan los canales de comunicación y dependencias entre servicios.

4

Generación

Una vez finalizado el diseño, genera el código base de tu arquitectura con un solo clic. La plataforma creará todos los archivos necesarios siguiendo las mejores prácticas de desarrollo.
